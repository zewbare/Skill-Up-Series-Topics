Technologies Covered in Training

Git

Git is a VCS created by Linus Torvalds in 2005 for the development of the linux kernel. It is used by developers for tracking changes in source code in a coordinating work such as software develeopment.

GitHub

Github is a company which provides a cloud-based Git repository hosting service. It offers the version control and SCM functionality of Git plus its own additional features such as access control, collaborations, bug tracking, feature requests etc.

MySQL

MySQL is an open source Relational Database Management System (RDBMS) based on Structured Query Language (SQL) which is the most popular language for accessing and manipulating content in a database. The most common use for MySQL is web database.

Django

Django is an open source python based web framework that follows the model-template-view (MTV) architectural pattern. It is used to ease the creation of complex, database-driven websites.

Flask

FFlask is an extensible micro web framework for building web application with Python. Unlike Django, flask is lightweight meaning it can be used for web develeopment of simple web applications by providing the required functionality.

Android Studio

Android Studio is the official IDE for Android OS built on JetBrains' IntelliJ IDEA. It is tailor-made for the develepment of Android apps with tools like including rich code, debugging, editing, testing and profiling tools.
